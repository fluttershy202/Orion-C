1.Penjelasan Apa itu Version Control

Kontrol versi (version control) adalah sistem manajemen kode sumber yang mencatat setiap perubahan pada file atau proyek seiring waktu. Alat ini memungkinkan pengembang melacak riwayat perubahan, membatalkan kesalahan, dan berkolaborasi secara efisien tanpa menimpa pekerjaan anggota tim lain.

2. Perbedaan Git dan Github

Git adalah perangkat lunak version control system (VCS) lokal untuk melacak perubahan kode di komputer Anda, sedangkan GitHub adalah layanan hosting berbasis cloud untuk menyimpan, mengelola, dan berkolaborasi pada repositori Git secara daring. Git bekerja offline, sementara GitHub memerlukan internet untuk berbagi dan berkolaborasi.

3. Penjelasan apa itu HTML

HTML (HyperText Markup Language) adalah bahasa standar berbasis tag yang digunakan untuk membuat struktur dan kerangka dasar halaman website. HTML mengatur tata letak elemen seperti teks, gambar, video, dan link agar dapat ditampilkan di browser internet.
